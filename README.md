# learnbuddy-personalized-learning-platform
Adaptive learning platform generate quizzes and learning modules based on learner classification (Slow/Intermediate/Fast).

Problem Statement

Traditional education and e-learning systems often lack customization, providing a one-size-fits-all approach that does not cater to individual learning speeds and styles. As a result, learners may struggle to stay engaged, retain information, or progress efficiently according to their abilities, leading to suboptimal learning outcomes.

Objectives

To create an adaptive learning platform that personalizes content for each student based on their learning speed (Slow/Intermediate/Fast). 
To automatically generate quizzes and learning modules tailored to individual learner classifications.
To enhance engagement and retention by providing targeted learning experiences for every user.

Technology Stack

    Frontend: HTML, CSS, JavaScript
    Backend: Flask
    Database: Firebase
